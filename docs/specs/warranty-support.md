## 5. Troubleshooting Procedures

### 5.1 End-User Troubleshooting

| Requirement ID | Symptom | User Steps | When to Contact Support |
|---|---|---|---|
| WS-TSH-001 | Unit not powering on | Check vehicle ACC is on; check if fuse is blown (refer to manual) | If fuse is intact and ACC is on |
| WS-TSH-002 | No audio output | Check volume level; check audio source; try different source | If no source produces audio |
| WS-TSH-003 | Phone not connecting via Bluetooth | Toggle phone Bluetooth off/on; forget device and re-pair | If pairing fails after 3 attempts |
| WS-TSH-004 | CarPlay/Android Auto not connecting | Try different USB cable; restart phone; check phone OS compatibility | If issue persists with known-good cable |
| WS-TSH-005 | GPS no signal | Drive to open area; check antenna connection (installer) | If no fix after 15 minutes in open area |
| WS-TSH-006 | Touchscreen unresponsive | Power cycle unit (ACC off, wait 30s, ACC on) | If unresponsive after power cycle |
| WS-TSH-007 | System freezing/rebooting | Check for available firmware update | If persists after update |
| WS-TSH-008 | Display flickering | Adjust brightness settings; check for nearby interference sources | If persistent across brightness levels |

### 5.2 Installer Diagnostic Procedures

| Requirement ID | Diagnostic Area | Procedure | Tools Required |
|---|---|---|---|
| WS-TSH-009 | Power supply verification | Measure voltage at harness connector (ACC on, engine running, cranking) | Multimeter |
| WS-TSH-010 | Speaker output test | Run 4-channel test tone from diagnostic app | Diagnostic app |
| WS-TSH-011 | CAN bus verification | Monitor CAN messages using built-in CAN monitor or external tool | Diagnostic app / CAN tool |
| WS-TSH-012 | GPS module test | Run GPS diagnostic (satellite count, signal strength) | Diagnostic app |
| WS-TSH-013 | Bluetooth module test | Reset Bluetooth, run discovery test, check antenna | Diagnostic app |
| WS-TSH-014 | Display diagnostics | Run pixel test, backlight test, touch calibration | Built-in service menu |
| WS-TSH-015 | Thermal status check | Read SoC temperature, ambient sensor, thermal throttle history | Diagnostic app |
| WS-TSH-016 | Full system diagnostic export | Export diagnostic package to USB for tech support review | USB drive |

## 6. Warranty Void Conditions

### 6.1 Actions That Void the Warranty

| Requirement ID | Condition | Scope of Void | Detection Method |
|---|---|---|---|
| WS-VOD-001 | Opening the sealed chassis (tamper seal broken) | Full hardware warranty voided | Tamper-evident seal inspection |
| WS-VOD-002 | Modification of internal components | Full hardware warranty voided | Internal inspection on RMA |
| WS-VOD-003 | Installation of unauthorized firmware | Full warranty voided | Firmware signature verification |
| WS-VOD-004 | Liquid immersion (beyond IP52 splash rating) | Full hardware warranty voided | Internal moisture indicators |
| WS-VOD-005 | Physical abuse (dropping, crushing) | Full hardware warranty voided | Physical inspection |
| WS-VOD-006 | Improper voltage supply (> 16V sustained, reverse polarity) | Full warranty voided | Internal fuse/protection circuit inspection |
| WS-VOD-007 | Use in non-specified vehicle without adapter validation | CAN bus-related issues excluded | Installation record review |
| WS-VOD-008 | Removal of safety labels or serial number | Full warranty voided | Visual inspection |

### 6.2 Actions That Do NOT Void the Warranty

| Requirement ID | Action | Warranty Status |
|---|---|---|
| WS-VOD-009 | Installation by uncertified installer | Warranty valid, but installation-related damage not covered |
| WS-VOD-010 | Installing OTA firmware updates | Warranty fully valid |
| WS-VOD-011 | Factory reset | Warranty fully valid |
| WS-VOD-012 | Normal cosmetic wear (minor scratches) | Warranty valid (cosmetic issues not covered regardless) |
| WS-VOD-013 | USB port normal wear | Warranty fully valid |
| WS-VOD-014 | Using third-party USB cables | Warranty valid (cable damage not covered) |

### 6.3 Tamper Detection

| Requirement ID | Mechanism | Specification |
|---|---|---|
| WS-VOD-015 | External tamper seal | Void-if-removed holographic seal on chassis seam |
| WS-VOD-016 | Internal moisture indicators | Humidity indicator strips inside chassis (visible on disassembly) |
| WS-VOD-017 | Firmware integrity log | Secure boot log records any unsigned firmware attempts |
| WS-VOD-018 | Hardware modification detection | PCB revision and component matching verified during RMA inspection |
